# Что такое TypeSpec

TypeSpec — язык от Microsoft для **описания API и моделей данных** в компактном
виде, из которого генерируются OpenAPI, JSON Schema, Protobuf и клиентский код.
По сути это удобный «исходник контракта», а OpenAPI — один из результатов
компиляции.

## Проблема, которую решает

Писать большие OpenAPI-спецификации в YAML руками больно: многословно,
повторяется, легко ошибиться. TypeSpec даёт **лаконичный язык** (похож на
TypeScript) с типами, моделями и переиспользованием, а генератор превращает его
в OpenAPI и другие форматы.

## Как выглядит

```typespec
model User {
  id: int64;
  name: string;
  email: string;
}

@route("/users")
interface Users {
  @get read(@path id: int64): User;
  @post create(@body user: User): User;
}
```

Компилятор из этого выдаёт полноценную OpenAPI-спецификацию (или Protobuf, JSON
Schema и т.д.).

## Зачем это, если есть OpenAPI

- **Компактность и типизация** — меньше ручного YAML, переиспользуемые модели.
- **Один источник → много форматов** — из одного описания получить OpenAPI для
  REST и Protobuf для gRPC.
- **Contract-first по-человечески** — контракт писать приятнее, чем сырой YAML.

## Место в связке

TypeSpec — это уровень **выше** OpenAPI: TypeSpec → (компиляция) → OpenAPI →
(генерация) → код/документация. Технология относительно новая и нишевая.

!!! note "Честно про опыт"
    С TypeSpec в проде не работал — знаю на уровне «что это и зачем»: компактный
    язык описания API, из которого генерится OpenAPI и клиенты. Детали
    промышленного применения не готов расписывать глубоко.

## Как ответить на интервью

Коротко: TypeSpec — язык от Microsoft для описания API и моделей данных,
похожий на TypeScript, из которого компилятор генерирует OpenAPI, JSON Schema,
Protobuf и код. Он решает боль ручного написания больших OpenAPI-спецификаций в
YAML: компактнее, типизировано, модели переиспользуются, а из одного описания
можно получить и REST-контракт, и gRPC. По сути это удобный источник для
contract-first, стоящий на уровень выше OpenAPI. Технология новая и нишевая.
