## Как это работает

- Всё содержимое — markdown-файлы в каталоге `docs/`
- Каждая новая страница добавляется в `nav` в файле `mkdocs.yml`
- Пуш в ветку `main` → сайт автоматически пересобирается на GitHub Pages

## Локальный запуск

```bash
pip install -r requirements.txt
mkdocs serve
```

Открыть <http://127.0.0.1:8000> — страница перезагружается при сохранении файла.
