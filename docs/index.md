# Подготовка к интервью

Конспект подготовки к интервью на позицию Java backend разработчика.

## Навигация

- [Содержание](contents.md) — список тем и разделов, план наполнения

## Как это работает

- Всё содержимое — markdown-файлы в каталоге `docs/`
- Каждая новая страница добавляется в `nav` в файле `mkdocs.yml`
- Пуш в ветку `main` → сайт автоматически пересобирается на GitHub Pages
