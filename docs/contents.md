# Содержание

Структура конспекта: разделы, темы внутри них и порядок разбора.

Разделы сгруппированы в шесть блоков. Подтемы внутри разделов определяются
отдельно, по мере наполнения — здесь пока только каркас.

## Java

| Раздел | Что внутри |
|---|---|
| Java Core | ООП, `Object` и его методы, `equals`/`hashCode`, исключения, дженерики, `Optional`, строки, примитивы и обёртки |
| Коллекции | Иерархия `Collection` и `Map`, внутреннее устройство, сложности операций, выбор структуры под задачу |
| Stream API | Лямбды и функциональные интерфейсы, промежуточные и терминальные операции, `Collectors`, когда стримы уместны |
| Многопоточность | Потоки, `synchronized` и `volatile`, гонки и deadlock, атомарные типы, пулы потоков, конкурентные коллекции |
| JVM и память | Стек и куча, жизненный цикл объекта, сборка мусора, `OutOfMemoryError` и `StackOverflowError`, JIT |

## Spring

| Раздел | Что внутри |
|---|---|
| Spring и Spring Boot | IoC и DI, бины и контекст, скоупы, автоконфигурация и стартеры, Spring MVC, AOP, `@Transactional`, профили и конфигурация |
| Экосистема Spring | Spring Security: аутентификация, авторизация, цепочка фильтров. Spring Cloud Gateway: маршруты, фильтры, роль шлюза |

## Данные

| Раздел | Что внутри |
|---|---|
| БД и SQL | Реляционная модель, `JOIN` и агрегация, индексы, транзакции и уровни изоляции, блокировки, план запроса, специфика PostgreSQL |
| Работа с БД из приложения | JPA и persistence context, Hibernate, EclipseLink, связи и ленивая загрузка, N+1, Spring Data JPA, миграции на Flyway и Liquibase |
| Не-реляционные хранилища | Redis: структуры данных, кэширование, инвалидация. Объектные хранилища: MinIO и S3, работа с файлами |

## Взаимодействие

| Раздел | Что внутри |
|---|---|
| HTTP и REST | HTTP: методы, коды, заголовки. REST: ресурсы, идемпотентность, statelessness, версионирование, JSON и Jackson |
| Остальные протоколы | gRPC и Protobuf, WebSocket, SSE, Long Polling — чем отличаются и что когда выбирать. SMTP и отправка почты |
| Kafka | Топики, партиции, группы потребителей, гарантии доставки, оффсеты. Avro и Schema Registry, эволюция схем |
| Контракты и документация API | OpenAPI и Swagger, TypeSpec, генерация кода и спецификации, contract-first против code-first |

## Инженерия

| Раздел | Что внутри |
|---|---|
| Тестирование | Виды тестов, JUnit, Mockito и моки, Testcontainers, тестирование Spring-приложений |
| Сборка и Git | Maven и Gradle: зависимости, фазы, плагины. Git: ветки, merge и rebase, конфликты, командный флоу |
| Проектирование | SOLID, основные паттерны, слои приложения, монолит против микросервисов |
| Другие языки и платформы | Kotlin и его отличия от Java, Quarkus, JavaEE — чем отличаются от Spring |
| Инструменты и библиотеки | Lombok, MapStruct, Thymeleaf, кодогенерация и annotation processing, рабочие инструменты |
